# Checklist Pengumpulan Tugas 1

Gunakan checklist ini sebelum submit Google Form.

## Dokumentasi Instalasi

- [ ] Topik yang dipilih dan alasan pemilihan topik.
- [ ] Pengaturan virtualisasi, VPS, dan sistem operasi Linux.
- [ ] Instalasi dan konfigurasi aaPanel beserta keamanan dasar.
- [ ] Pengaturan Cloudflare Tunnel dan akses publik platform.
- [ ] Vibe-coding, deployment, frontend, backend, dan database.
- [ ] Tech stack dan layanan yang digunakan beserta alasan.
- [ ] Penjelasan singkat setiap komponen.
- [ ] Gambar relevan dan jelas.

## Link dan Screenshot

- [ ] Link platform publik sudah bisa dibuka dari jaringan luar.
- [ ] Screenshot browser menampilkan URL publik.
- [ ] Screenshot antarmuka HiddenGem Explorer.
- [ ] Screenshot Cloudflare Tunnel `HEALTHY`.
- [ ] Screenshot aaPanel site configuration.

## Validasi Teknis

- [ ] `npm install` berhasil.
- [ ] `npm run lint` berhasil.
- [ ] `npm run build` berhasil.
- [ ] Folder `dist/` tersedia.
- [ ] File deployment sudah dipindahkan ke folder site aaPanel.

## Catatan Rahasia

- [ ] Password aaPanel tidak dimasukkan ke laporan.
- [ ] Token Cloudflare Tunnel tidak dimasukkan ke laporan.
- [ ] API key tidak dicantumkan di repo publik.
